[Eng]Automated system "Keyboard simulator".

	Target: Training program, teaches you to type English text on the keyboard, without looking at the keyboard

	Description: Supports learning at several levels (the middle row of the keyboard, then the top row is added, and the entire keyboard).
allows you to adapt any English text to the desired level of the student.
save the text "F5" at this stage and continue later "F9",
Outputs the result "F1" or when the end of the text is reached,
(typing speed and accuracy percentage in typing).

	Development: support library folder "C:\library" (with .txt files) from anywhere on PC or go to database,
support for the learning mode for the Russian layout, a table of results, analysis and output of the most common errors,
tutorial drawings: the position of the fingers on the keyboard.

[Rus]Автоматизированная система "Тренажёр клавиатуры".
	Цель: Обучающая программа, обучает набирать английский текст на клавитуре, не смотря на клавиатуру.

	Описание: Поддерживает обучение на нескольких уровнях (средний ряд клавиатуры, далее добавляется верхний ряд, и вся клавиатура целиком).
позволяет адаптировать любой английский текст в нужному уровню обучающегося.
сохранить текст "F5" на данном этапе и продолжить позже "F9",
Выводит результат  "F1" или по достижении конца текста,
(скорость набора текста и процент точности в наборе слов).

	Планы: поддержка папки library "C:\library" (с .txt файлами) из любого места на ПК или переход на базу данных, 
поддержка режима обучения для русской раскладки, таблица учёта результатов, анализирование и вывод наиболее частых ошибок,
обучающие рисунки: положение пальцев на клавиатуре.

	ВНИМАНИЕ: для корректной работы программы папку c текстами "library" разместить по адресу "C:\library".
